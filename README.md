# Smart Compost Flask Application

A web application for monitoring and managing a smart compost system.

## Features

- Dashboard with real-time compost metrics
- Network configuration for IoT connectivity
- Analytics and statistics visualization
- User profile management

## Installation

1. Clone the repository
2. Create a virtual environment:
   ```
   python -m venv venv
   ```
3. Activate the virtual environment:
   - Windows: `venv\Scripts\activate`
   - macOS/Linux: `source venv/bin/activate`
4. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
5. Run the application:
   ```
   python app.py
   ```
6. Open your browser and navigate to `http://127.0.0.1:5000/`

## Project Structure

- `app.py` - Main Flask application
- `templates/` - Jinja2 HTML templates
- `static/` - CSS, JavaScript, and other static files
